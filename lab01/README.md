# Лабораторная работа №1
## Типы данных,память и выражения в C

## Цель работы
Научиться работать с базовыми типами данных языка C, понимать связь переменных 
и памяти, использовать операции и выражения, выполнять ввод и вывод данных, а 
также понимать переполнение и точность вычислении.

## Задание 1. Первая программа с переменными
Создаите программу, которая объявляет переменные int a, float b, char c. Присвоите 
им значения и выведите их на экран.
Пример вывода: a = 10
b = 3.14
c = A

**Решение:** task1.c 
**Инструкция по запуску:**
```
gcc task1.c -o task1
./task1
```
