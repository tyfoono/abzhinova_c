# Вопросы
- Как объявить структуру в языке C?

```
struct имя_структуры {
    объявляются переменные, которые являются полями
    например int age; 
}
```

- Что такое вложенные структуры и зачем они нужны?

Вложенные структуры - структуры, полями которых являются другие структуры. 
Они нужны в случае, когда нужны не стандартные типы данных для информации внутри другой структуры.

- Как передать структуру в функцию?

```
void func(struct имя_структуры имя_аргумента) {...}
```

- Можно ли создавать массивы структур?

Можно. 

```
struct имя_структуры имя _массива[];
```

- Как обратиться к элементу массива структур?

Так же, как и к обычному элементу массива...

```
имя_массива[i]
```

или через указатель

- Можно ли использовать enum внутри структуры?

Можно 

- Можно ли сравнивать структуры оператором ==?

Нельзя. Нужно сравнивать их поля отдельно.
