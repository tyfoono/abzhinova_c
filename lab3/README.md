## Вопросы и ответы

### 1. Что делает malloc?
Выделяет блок памяти заданного размера для массива

### 2. Чем отличается malloc от calloc?
- malloc — выделяет память, но не записывает значения
- calloc — выделяет память и записывает нули

### 3. Что делает realloc?
Изменяет размер ранее выделенного блока памяти. Может увеличить или уменьшить его, при необходимости перемещая данные в новое место

### 4. Что произойдет при утечке памяти?
Программа будет потреблять память без освобождения, что может привести к замедлению работы, исчерпанию памяти и аварийному завершению

### 5. Зачем нужен free?
Освобождает динамически выделенную память, возвращая её системе. Предотвращает утечки памяти

### 6. Что произойдет при двойном освобождении памяти?
Неопределенное поведение. Программа может аварийно завершиться или повредить память

### 7. Как проверить, что malloc вернул NULL?

```
int *arr = (int*)malloc(n * sizeof(int));
if (arr == NULL) {
    printf("Ошибка выделения памяти!\n");
}
```

### 8. В чем отличия статическго и динамического массива?

Статический массив выделяется в памяти один раз и не может изменять размер 
У динамического массива можно изменять размер
