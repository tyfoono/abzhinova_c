## Ответы на вопросы

### 1. Что такое файл в языке C?

Набор данных, хранящийся на диске

### 2. Что такое FILE*?

FILE* — это указатель на структуру, в которой операционная система хранит всю информацию об открытом файле

### 3. Какие режимы открытия файла существуют?

Основные режимы:


`"r"` - открыть существующий файл для чтения
`"w"` - создать новый файл для записи (старое содержимое стирается)
`"a"` - открыть файл для добавления в конец
`"r+"` - открыть для чтения и записи
`"w+"` - создать для чтения и записи (старое содержимое стирается)
`"a+"` - открыть для чтения и добавления в конец

Добавление буквы `"b"` означает бинарный режим.

### 4. Чем отличается текстовый файл от бинарного?

Текстовый файл содержит только символы из таблицы ASCII - буквы, цифры, знаки препинания. Каждая строка обычно заканчивается специальным символом перевода строки

Бинарный файл содержит данные в байтах

### 5. Что делают функции putc и getc?

putc — записывает один символ в файл

getc — читает один символ из файла

### 6. Что делают fgets и fputs?

fputs - записывает строку в файл

fgets — читает строку из файла с ограничениями

### 7. Что делают fprintf и fscanf?

fprintf — работает точно как printf, но в файл

fscanf — работает как scanf, но читает из файла

### 8. Что делают fread и fwrite

fwrite - записывает в файл блок данных так, как они лежат в памяти

fread - читает блок данных из файла в память

### 9. Что делают fseek и ftell

fseek - перемещает курсор внутри файла в нужное место

ftell - возвращает на каком месте сейчас находится курсор

### 10. Что делают feof и ferror?

feof - проверяет дошёл ли курсор до конца файла

ferror — проверяет, произошла ли ошибка при работе с файлом

### 11. Почему важно закрывать файл?

При записи данные буферизируются. fclose гарантированно сохраняет их на диск, так данные не протеряются
Также открытые файлы расходуют лимиты ОС. Если не закрывать, можно исчерпать доступные дескрипторы и потерять возможность открывать новые файлы
